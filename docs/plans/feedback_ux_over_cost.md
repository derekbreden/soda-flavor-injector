---
name: UX is king, cost is no concern
description: When making design decisions, user experience is the primary criterion; cost is never a tiebreaker or deciding factor
type: feedback
---

UX is the primary concern for all design decisions — above durability, simplicity, prototypability, and cost. Cost is explicitly not a concern.

**Why:** The user called this out after a decision agent chose a hardware-store-bolt approach (3 turns, worse UX) over a 3D-printed approach (half turn, better UX) primarily because the bolt was "mechanically certain" and "debuggable." The user disagreed — the printed approach had adequate durability (100+ cycles vs 36 needed) and dramatically better UX. The decision was reversed.

**How to apply:** When comparing design alternatives, weight UX (one-handed operation, intuitive feel, speed, dark-cabinet usability) above all other criteria. Never use cost as a reason to choose one approach over another. If durability is adequate (survives the expected lifecycle with reasonable margin), don't penalize an approach for having less margin than an alternative. Only reject a UX-superior approach if it is mechanically infeasible or has durability below the required lifecycle.
